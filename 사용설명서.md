# 🌷 꽃집 매입장부

꽃시장에서 받은 수기 영수증을 즉시 정리하고, 월말에 세금계산서 발행을 요청하기 위한 휴대폰용 앱입니다.

---

## 무엇이 만들어졌나

| 파일 | 설명 |
|---|---|
| `index.html` | 앱 본체 (이 파일 하나로 모든 기능이 동작) |
| `manifest.webmanifest` | 홈 화면 앱 아이콘 설정 |
| `sw.js` | 오프라인 동작용 (인터넷 약해도 사용 가능) |
| `icon-192.png` / `icon-512.png` | 홈 화면 아이콘 |

- **데이터는 사용하는 기기(브라우저) 안에 저장**됩니다. 계정·서버 불필요, 무료.
- 한 기기에서만 보입니다. 기기를 바꾸거나 정리할 땐 **거래처 탭 → 데이터 백업**으로 파일을 저장해두세요.

---

## 기능 요약

### ✏️ 입력 (현장에서 3초 입력)
- 거래처는 **이름** 또는 **업체번호**로 입력 → 기존 거래처는 자동완성(자주 쓰는 곳 우선).
- 처음 보는 거래처면 번호(1~9999)와 이름만 정하면 자동 등록.
- 금액 입력 시 천단위 콤마 + `金 백오십오만원정` 한글 표기 자동 표시.
- 날짜는 오늘로 자동 입력(영수증 날짜가 다르면 바꾸면 됨).

### 📋 내역
- 이번달·지난달·올해·전체·직접선택 기간별 매입 내역과 합계.
- 잘못 입력한 건 삭제 가능.

### 📊 대시보드
- 선택 기간 **총 매입 금액**.
- **업체별 매입 순위 그래프**(막대 + 순위).

### 🏪 거래처
- 거래처별 누적 매입액, 검색.
- 거래처 선택 → **기간 선택 후 `전자세금계산서 발행정보` 이미지 생성**
  - 거래처명 + 선택 기간 합계가 들어간 양식 이미지를 만들어
  - **카톡 등으로 바로 공유**하거나 이미지로 저장 → 계산서 발행 요청용으로 전달.
- **데이터 백업 / 복구**, **엑셀용 CSV 내보내기**(월말 정산·세무용).

> 발행 정보 양식의 사업자 정보(그래머시플라워 성의회관점 / 641-91-02374 등)는 고정으로 들어갑니다.

## 휴대폰에 설치하기 (한 번만)

이미 GitHub Pages로 배포되어 **고정 주소**가 있습니다:

### 👉 https://jadekim-readians.github.io/flower-ledger/

1. 위 주소를 휴대폰으로 엽니다 (카톡 "나에게 보내기" 등으로 링크 전달)
2. **홈 화면에 추가**
   - 아이폰(Safari): 공유 버튼 ⬆️ → "홈 화면에 추가"
   - 안드로이드(Chrome): 메뉴(⋮) → "앱 설치" 또는 "홈 화면에 추가"
3. 홈 화면의 🌷 아이콘으로 앱처럼 사용

## 자동 업데이트

- 앱 코드(`index.html` 등)를 고친 뒤 **GitHub에 push**하면, 1~2분 내 위 주소가 자동 갱신됩니다.
- 저장소: https://github.com/JadeKim-Readians/flower-ledger
- 보통은 "○○ 기능 고쳐줘"라고 말씀해주시면 제가 수정 후 push까지 해드립니다.
- 휴대폰 앱이 옛 화면이면, 앱을 껐다 켜면 새 버전으로 갱신됩니다.

- **여러 기기에서 같이 쓰고 싶다** → 지금은 기기별 저장입니다. 클라우드 동기화가 필요하면 말씀해주세요(별도 작업).
